# Rule 10THB — Income Tax Rules, 1962

**Title:** Eligible specified domestic transaction

---

**Eligible specified domestic transaction.**

**10THB.** The "eligible specified domestic transaction" means a specified domestic transaction undertaken by an eligible assessee and which comprises of :—

|  |  |  |
| --- | --- | --- |
| (*i*) |  | supply of electricity 47;)[\*\*\*]; or |
| (*ii*) |  | transmission of electricity; or |
| (*iii*) |  | wheeling of electricity; 48;)[or |
| (*iv*) |  | purchase of milk or milk products by a co-operative society from its members.] |

---

47. Words "by a generating company" omitted by the IT (Second Amdt.) Rules, 2016, w.e.f. 17-2-2016.

48. Inserted by the IT (Nineteenth Amdt.) Rules, 2015, w.e.f. 8-12-2015.
